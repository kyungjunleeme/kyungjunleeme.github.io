# Medipartner AI_demosite
Medipartner가 보유한 AI 알고리즘을 적용을 확인할 수 있는 사이트입니다.

# How to use

1. pip install virtualenv

2. virtualenv [가상환경명]

3. call [가상환경명]/scripts/activate (가상환경 activate)

4. pip install -r requirements.txt

5. **Run "main.py"**

6. 127.0.0.1:5000 접속


# Arguments
usage:
```
python server.py
```
